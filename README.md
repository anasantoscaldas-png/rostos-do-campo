# 🌾 Pontes de Afeto — Do Campo à Cidade

**Pontes de Afeto** é uma plataforma digital desenvolvida para conectar realidades que, embora dependentes, muitas vezes parecem distantes: o cotidiano do produtor rural e a rotina dos grandes centros urbanos.

O grande diferencial deste projeto é mudar o foco do consumo para o lado humano. Em vez de focar apenas na logística, na comercialização ou nas métricas de produtos agrícolas, a plataforma atua como um mural vivo de relatos, focado nas **histórias, rostos, dores, orgulhos e sonhos** de quem trabalha a terra para alimentar a cidade.

### 🎯 Propósito do Projeto

* **Humanizar o Consumo:** Mostrar que por trás de cada alimento na mesa urbana existe uma trajetória de vida.
* **Criar Empatia:** Quebrar a barreira invisível que resume o campo a gôndolas de supermercado e códigos de barra.
* **Dar Voz:** Oferecer um espaço de protagonismo para a agricultura familiar, comunidades tradicionais e iniciativas sustentáveis.

### 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando uma abordagem moderna, leve e totalmente responsiva:

* **HTML5** para a estruturação semântica das páginas.
* **Tailwind CSS** para um design sofisticado, limpo e focado na experiência de leitura.
* **CSS3 Customizado** para animações sutis de transição e refinação estética.
* **JavaScript (Vanilla)** para a modelagem dos dados e renderização dinâmica dos cartões de histórias.
